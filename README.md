# 🌍 SiteDeViagem

Projeto desenvolvido durante minha formação Full Stack na Recode Pro.

O sistema foi criado com foco em simular uma plataforma de viagens, utilizando HTML, CSS, JavaScript e Bootstrap.

## 🚀 Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- Bootstrap

## 🎯 Objetivos do Projeto

- Praticar desenvolvimento web responsivo
- Aprender estruturação de páginas
- Trabalhar estilização moderna
- Evoluir conceitos de JavaScript
- Criar experiência visual inspirada em plataformas de viagem
